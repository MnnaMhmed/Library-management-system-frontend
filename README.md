# Library Management System - Frontend

This is the **frontend** of the Library Management System built with **Angular**.

## Features
- Browse and search books
- Manage book inventory
- Add, update, and delete books
- Responsive UI

## Technologies Used
- Angular
- TypeScript
- HTML & CSS
- Bootstrap (optional)
